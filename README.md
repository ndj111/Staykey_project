## :purple_circle: Description
풀빌라, 독채, 펜션, 글램핑 등 SNS에서 자주 찾아볼 수 있는 감성숙소에 대한 수요는 늘었으나, 정보가 산재해 있어 한 번에 찾기 어려움.

이에 따른 국내 여행 숙소에 대한 소개/예약 사이트. 클론 코딩.

크게 사용자 화면과 관리자 화면으로 구성.

<br/>
